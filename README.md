# 🎮 NeonArcade

Willkommen bei **NeonArcade**! Hier findest du eine wachsende Sammlung selbst programmierter Minispiele, die mit **HTML, CSS und JavaScript** entwickelt wurden. Das Projekt dient dazu, meine Programmierkenntnisse zu verbessern und immer neue Spiele sowie Features hinzuzufügen.

## 🕹️ Aktuelle Spiele

### 🔢 Number Guessing Game

Errate die zufällig generierte Zahl in möglichst wenigen Versuchen. Wähle zwischen verschiedenen Schwierigkeitsgraden, sammle Highscores und verfolge deine Statistiken.

### ⚡ Reaction Game

Teste deine Reaktionsgeschwindigkeit! Warte auf das Signal und klicke so schnell wie möglich. Zu frühes Klicken zählt als Fehler. Versuche, deinen Highscore zu schlagen.

### 🐍 Snake

Steuere die Schlange, sammle Äpfel und werde immer länger. Vermeide Kollisionen mit deinem eigenen Körper und stelle einen neuen Highscore auf.

### 🎯 Aim Trainer

Trainiere deine Zielgenauigkeit und Reaktionsgeschwindigkeit. Triff die zufällig erscheinenden Ziele innerhalb der Zeit und sammle möglichst viele Punkte.
